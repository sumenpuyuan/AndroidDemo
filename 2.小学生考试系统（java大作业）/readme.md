##java大作业 学生考试系统##

实现功能很简单，学生答题，给出分数，结果，还可以查看之前的做题记录。

数据库相关文件目录是自己查数据库时的资料，当资料来看。

数据库有四个表：用户表，题库表，粗略记录表，详细记录表。
